# 🛒 Product Management System

A simple **Spring Boot web application** to manage products with CRUD operations, pagination, and search functionality.

---

## 🚀 Features

* ➕ Add Product
* 📋 View Products
* ✏️ Update Product
* ❌ Delete Product
* 🔍 Search Products
* 📄 Pagination
* ✅ Form Validation

---

## 🛠️ Tech Stack

* Java 21
* Spring Boot
* Spring Data JPA
* MySQL
* Thymeleaf
* Tailwind CSS

---

## 📂 Project Structure

```
controller/
service/
repository/
entity/
exception/
templates/
```

---

## ▶️ How to Run

1. Clone the repository
2. Open in IDE (Eclipse / IntelliJ)
3. Configure MySQL database in `application.properties`
4. Run the project
5. Open browser → http://localhost:8080
